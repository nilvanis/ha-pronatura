# ProNatura

Monitor your garbage collection plan directly from ProNatura without scraping PDFs.

## Highlights

- Guided config flow with searchable street/addresses direct from the ProNatura API
- Date sensors for every fraction with metadata attributes
- Diagnostics download, Repairs reconfigure, and daily refresh handled by a DataUpdateCoordinator

## Installation

1. Install via HACS and restart Home Assistant.
2. Add the **ProNatura** integration from *Settings → Devices & Services*.
3. Pick your street/address combination – sensors for every fraction are created automatically.
